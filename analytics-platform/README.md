# Analytical Platform
A full-stack web application for visualizing and analyzing time-series measurement data.

## Features

- **Dynamic Visualization**: Supports multiple chart types (Line, Bar, Pie, Polar Area) using **Chart.js**.
- **Data Analytics**: Calculates real-time metrics including **Average, Minimum, Maximum, and Standard Deviation**.
- **Time-Series Filtering**: Filter data by specific fields and date ranges.
- **Robust Error Handling**: Informative alerts for invalid date ranges or missing data (e.g., searching in 2010).
- **Clean Architecture**: Separation of concerns with dedicated API routes and modular frontend files.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+), Chart.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Environment**: dotenv for configuration

## Getting Started

### 1. Prerequisites
- Node.js installed
- MongoDB instance (local or Atlas)
